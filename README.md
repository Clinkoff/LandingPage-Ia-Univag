# Lenovo Landing Page (MVP)

Uma Landing Page desenvolvida para uma aula da univag

## Tecnologias Utilizadas

- **HTML5 Semântico**: Estrutura otimizada.
- **Tailwind CSS v4**: Estilização moderna sem arquivo de configuração JS (`@theme` direto no CSS).
- **JavaScript (Vanilla)**: Lógica para slider, menu mobile e animações de scroll (Intersection Observer).

## Estrutura do Projeto

```plaintext
lenovo-landing-page/
├── assets/          # Imagens (Logos, Produtos, Heros)
├── src/
│   ├── css/
│   │   └── input.css  # Configuração do Tailwind v4 (@theme)
│   └── js/
│       └── main.js    # Lógica do Slider e Animações
├── dist/            # CSS compilado final
└── index.html       # Página principal
```
